## Windows+Android操作步骤
```
第一步：下载python最新版本，下载地址https://www.python.org/ ，需要将python加入环境变量
```
```
第二步：运行 pip install -r requirements.txt
```
```
第三步：下载adb驱动，下载地址https://adb.clockworkmod.com/
```
```
第四步：运行adb devices查看是否安装成功
```
```
第五步：运行python wenda.py 百万英雄 
```
